# マーケティング戦略メモ

## 現状の課題
セグメントが定義されておらず、新規機能開発の優先順位が定まっていない。リファクタリングが終わるまでに整理しておきたい
色々やろうとして、みんなに使ってほしくて、対象を広げすぎて解像度が下がり、結果として誰にも強く刺さらない状態になっていないか？

まずはとにかく範囲を絞ってこういう人には刺さるんよ！と友達に紹介できる形にしたい

## 実際に反応した層

あと、を紹介したnote記事に反応したのは非常に限定的な層だった。

個人開発者、エッセイスト、表現や執筆体験に強いこだわりを持つ書き手...

いずれも徳永に近い、新しいツールを積極的に試す傾向があり、いわゆるアーリーアダプター特性を持つユーザーならとりあえずそこに合わせていくしかない

### 方針

1. アーリーアダプターに集中する

まずはすでに反応している層に最適化する。（徳永か？）
最初の目標は「多くの人に理解されること」ではなく、「特定の人に強く支持されること」。

2. そいつらに勝手に宣伝させる

アーリーアダプターは新しいものをいち早く使うこと自体に価値を感じる傾向がある。その心理を前提に設計したい。
使っていることが語りたくなるとか、使っている自分を見せたくなるような早期ユーザーであることが一種のステータスになる形にしたい
